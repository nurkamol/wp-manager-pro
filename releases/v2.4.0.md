# WP Manager Pro v2.4.0 — Dev Tools

**Released:** 2026-03-16

---

## Highlights

A dedicated 5-tab developer toolkit — edit `wp-config.php` visually, edit `.htaccess` safely, inspect PHP configuration, monitor live database queries, and tag your environment type with a coloured sidebar badge.

---

## New Features

### ⚙️ wp-config.php Visual Editor
- Constants grouped into categories: Database, Debug, Salts, URLs, Memory, Custom
- Boolean constants rendered as toggle switches; strings as text inputs
- `DB_PASSWORD` masked with an eye toggle to reveal
- **Regenerate All Salts** — fetches fresh keys from `api.wordpress.org/secret-key/1.1/salt/` and writes them to `wp-config.php` in one click
- Only modifies individual `define()` lines; never rewrites the whole file

### 🔧 .htaccess Editor
- Full textarea editor with file info bar (size, last modified, writable status)
- **Auto-backup** — saves a `.htaccess.wmp-backup` copy before every write
- **Restore from Backup** button — one-click rollback to last backup

### 🔍 PHP Info Viewer
- Lazy-loaded on demand to avoid slowing down the page
- Search/filter across all PHP sections in real-time
- Collapsible accordion tables per section
- Amber highlight when **Local Value ≠ Master Value**

### 📊 Query Monitor (Lite)
- Reads `$wpdb->queries` when `SAVEQUERIES` is enabled
- Stat cards: total queries, total time, slow query count (> 50 ms), peak memory
- Query table with SQL, caller, time; slow queries highlighted in amber
- "Show slow only" toggle; sortable table

### 🌍 Environment Badge
- Select **Production / Staging / Development / Local** from colour-coded cards
- Reads `WP_ENVIRONMENT_TYPE` constant or saved plugin option (plugin option takes precedence)
- Coloured badge displayed in the sidebar header below the plugin name
- Badge colours: Production = red, Staging = orange, Development = green, Local = blue

## REST API — New Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/dev-tools/wp-config` | Parse `wp-config.php` constants into grouped structure |
| POST | `/dev-tools/wp-config` | Update or add a single `define()` constant |
| GET | `/dev-tools/htaccess` | Read `.htaccess` and `.htaccess.wmp-backup` |
| POST | `/dev-tools/htaccess` | Save `.htaccess` (auto-backup before write) |
| GET | `/dev-tools/phpinfo` | Return `phpinfo()` output parsed into sections |
| GET | `/dev-tools/query-monitor` | Return `$wpdb->queries` with stats |
| GET | `/dev-tools/environment` | Return environment type (constant or option) |
| POST | `/dev-tools/environment` | Save custom environment type override |

---

## Technical Notes

- `class-dev-tools-controller.php` — 8 public methods
- `wp-config.php` parser uses regex line-by-line replacement (no full file rewrite)
- PHP Info sections parsed from `ob_start()` / `phpinfo()` HTML output via `DOMDocument`
- Environment badge rendered in `Sidebar.tsx` via a dedicated `useQuery` to `/dev-tools/environment`

---

## Upgrade Notes

No database migrations required. Navigate to **WP Manager → Dev Tools** after activating.
