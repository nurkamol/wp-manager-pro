# WP Manager Pro v2.3.1 — Redis Drop-in & Maintenance Fixes

**Released:** 2026-03-16

---

## Highlights

Patch release bundling the Redis object-cache drop-in directly into the plugin (no external plugin needed), adding a Redis node to the WP admin bar, and fixing several maintenance-mode bugs discovered after v2.3.0.

---

## New Features

### 🔴 Bundled Redis Object-Cache Drop-in
- `includes/object-cache.php` ships with the plugin — install it to `wp-content/object-cache.php` with one click from **Performance → Object Cache**
- No third-party Redis plugin required; works with the PhpRedis PHP extension
- Enable / Disable / Flush actions available at any time

### 🔴 Redis Admin Bar Node
- Green pulsing dot + Redis version badge in the WP admin bar when the drop-in is active
- **Flush Cache** sub-item — AJAX flush with toast notification
- **Object Cache Settings** sub-item — links directly to Performance → Object Cache tab

### 🔧 Maintenance Improvements
- New **Admin Bar Toggle Visibility** setting — hide the maintenance toggle from the admin bar (hidden by default on fresh installs)
- **Custom Bypass URL Slug** — configure a secret `?{slug}=1` query parameter to preview the live site while maintenance mode is active

---

## Bug Fixes

- **Maintenance settings not persisting** — boolean options (`show_badge`, `show_countdown`, `show_adminbar_toggle`) were being stored as PHP booleans instead of `0`/`1` integers; `get_option` was returning inconsistent values after save
- **JS/CSS cache busting** — assets now versioned by `filemtime()` instead of a static constant; browser always fetches the latest build after a deploy
- **Page reload after maintenance save** — admin page now reloads ~800 ms after saving maintenance settings so the admin bar immediately reflects the new toggle visibility state

---

## REST API

No new endpoints. Existing `/maintenance/settings` endpoint extended to accept `show_adminbar_toggle` and `bypass_slug` fields.

---

## Upgrade Notes

No database migrations required. After updating, visit **Performance → Object Cache** to install the bundled Redis drop-in if desired.
