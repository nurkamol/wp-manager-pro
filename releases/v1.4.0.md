## Technical Notes

- **New REST routes**: 18 new endpoints added across `/snippets`, `/redirects`, `/email/*`, `/backup/*`, `/audit/*`
- **New PHP controllers**: `class-snippets-controller.php`, `class-redirects-controller.php`, `class-email-controller.php`, `class-backup-controller.php`, `class-audit-controller.php`
- **New DB tables**: `wp_wmp_snippets`, `wp_wmp_audit_log` (created on activation via `dbDelta`)
- Snippets toggle bug: root cause was `$wpdb->get_results()` returns all values as PHP strings â€” `(int)` cast added in `get_snippets()`, `create_snippet()`, and `update_snippet()`
- Redirect wildcard matching uses `fnmatch()` server-side
- Backup `.htaccess`: `Deny from all` / `Require all denied` dual-syntax for Apache 2.2 & 2.4 compatibility
