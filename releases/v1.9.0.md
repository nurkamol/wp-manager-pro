## What's New

### ⚡ Performance Page (new Tools section entry)
- New dedicated **Performance** page with three tabs: Overview, Transients, and DB Cleanup
- Sidebar icon: Gauge (lucide-react)

### 🗑️ DB Cleanup
- Select any combination of cleanup types with live item counts:
  - Post Revisions (with estimated KB saved)
  - Auto-Drafts
  - Trashed Content
  - Spam Comments
  - Pending Comments
  - Orphaned Post Meta
  - Orphaned Comment Meta
  - Expired Transients
- Confirmation dialog lists exactly what will be deleted before executing

### 🕐 Transient Manager
- Paginated table of all transients (50 per page) with search
- Shows: name, size in bytes, expiry timestamp, expired badge
- Delete individual transients with one click
- **Purge Expired** button clears all timed-out transients (regular + site transients) in one shot

### 🖥️ Object Cache Status
- Auto-detects Redis, Memcached, or any external object cache via `wp_using_ext_object_cache()`
- Displays a prominent green banner when cache is active (with type), or a grey warning with an install recommendation when not active

### REST API (5 new endpoints)
- `GET /performance/overview` — all counts + cache status
- `GET /performance/transients` — paginated transient list with search
- `DELETE /performance/transients` — delete single transient by name
- `POST /performance/transients/purge-expired` — delete all expired
- `POST /performance/cleanup` — run selected cleanup types
