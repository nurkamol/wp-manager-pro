## What's New

This is the **initial public release** of WP Manager Pro — a full-stack WordPress management plugin that replaces the need for multiple separate admin tools with a single, fast, modern React-powered interface.

---

## Highlights

### 11 Management Modules
Every module is accessible from a persistent sidebar and communicates with a dedicated REST controller. All routes require `manage_options` — safe for production, agency-ready by design.

| Module | What you can do |
|--------|----------------|
| **Dashboard** | Real-time site health: versions, update counts, memory, disk, DB size |
| **Plugin Manager** | Activate · Deactivate · Delete · Search & Install from WP.org |
| **Theme Manager** | Activate · Delete · Search & Install from WP.org |
| **File Manager** | Browse filesystem · Edit code files · Create folders · Delete files |
| **Database Manager** | Browse tables · View data · Search & Replace · Optimize · Run queries |
| **User Manager** | List · Change roles · Login As · Delete |
| **System Info** | WordPress · PHP · Database · Server · Constants · Cron |
| **Maintenance Mode** | Toggle on/off · Custom title & message |
| **Debug Tools** | Toggle WP_DEBUG · View error log · Clear log |
| **Image Tools** | WebP · JPEG quality · Max dimensions · Regenerate thumbnails |
| **Notes** | Color-coded, persistent note-taking |

## REST API

39 endpoints under `wp-manager-pro/v1`, all nonce-authenticated and capability-checked.

---

## Security Features

- All REST routes require `manage_options` + WordPress nonce (`X-WP-Nonce`)
- File Manager: `realpath()` path traversal protection, 2 MB read cap
- Write-protected files: `wp-config.php`, `.htaccess`, `index.php`
- Login-As tokens: single-use, 5-minute TTL via WordPress transients
- SQL runner: read-only (`SELECT`, `SHOW`, `DESCRIBE`, `EXPLAIN` only)
- All inputs sanitized (`sanitize_text_field`, `absint`, `wp_kses_post`)

## Requirements

- WordPress 5.9+
- PHP 7.4+
- MySQL / MariaDB 5.6+
- Modern browser (Chrome, Firefox, Edge, Safari)

---

## Installation

1. Download `wp-manager-pro.zip` from this release
2. **WP Admin → Plugins → Add New → Upload Plugin**
3. Upload the ZIP → **Install Now** → **Activate**
4. Open **WP Manager** in the admin sidebar

## Known Limitations

- File editor max size: 2 MB
- SQL runner is intentionally read-only (no INSERT/UPDATE/DELETE via UI)
- WebP conversion requires GD or ImageMagick to be compiled into PHP
- Login-As tokens expire after 5 minutes and are single-use
