# WP Manager Pro v1.7.0

## What's New

### Monaco Editor in Code Snippets
The Code Snippets create/edit dialog now uses the full **Monaco editor** (the same engine that powers VS Code) instead of a plain textarea. You get real syntax highlighting for PHP, CSS, and JavaScript, line numbers, word-wrap, and a properly sized editor that adapts to the dialog.

### Scheduled Backups
Set-and-forget automatic database backups. Choose a frequency (Daily / Weekly / Monthly), configure how many backups to retain, and WP Manager Pro handles the rest via WordPress Cron. Old backups beyond the retention limit are pruned automatically after each run.

### Server Config Generator
A new **Server Config Generator** card in Image Tools generates ready-to-copy server configuration snippets for:
- **WebP Serving** — Nginx `try_files` rule and Apache rewrite rule
- **Security Headers** — `X-Content-Type-Options`, `X-Frame-Options`, `X-XSS-Protection`, `Referrer-Policy`, `Permissions-Policy`
- **Browser Cache Rules** — long-lived caching for static assets

One click copies any snippet to your clipboard.

### White-label / Branding
A new **Settings** page lets you customise how WP Manager Pro appears in the WordPress admin:
- **Plugin Name** — shown in the sidebar header (e.g. replace "WP Manager Pro" with your agency name)
- **Admin Menu Label** — the text in the WordPress left-hand menu
- **Custom Logo URL** — replaces the default blue icon with your own image

Changes take effect immediately after saving; the menu label updates on the next page load.

## Changes
- `Sidebar.tsx` logo and plugin name are now driven by the saved branding options
- `wp_localize_script` exposes `branding.{ pluginName, menuLabel, logoUrl }` to the React app
- New `class-settings-controller.php` REST controller (`GET/POST /settings`)
- New `GET/POST /backup/schedule` REST routes
- WP Cron hook `wmp_run_scheduled_backup` registered; custom `monthly` recurrence added
- Version bumped: `1.6.0` → `1.7.0`
