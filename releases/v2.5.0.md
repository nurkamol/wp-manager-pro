# WP Manager Pro v2.5.0 — Command Palette & Settings Export/Import

**Released:** 2026-03-16

---

## Highlights

Global `Cmd+K` command palette for instant navigation and quick actions, a signed JSON settings export/import system for migrating configuration between sites, and a WordPress XML content export directly from the plugin.

---

## New Features

### ⌨️ Command Palette
- Open with **Cmd+K** / **Ctrl+K** (or the keyboard icon in the sidebar footer)
- Fuzzy search across all 25 pages — navigate instantly without using the sidebar
- **Quick Actions** available without leaving the current page:
  - Flush Object Cache
  - Toggle Maintenance Mode
  - Clear Error Log
  - Purge Expired Transients
  - Create Database Backup
- **Recent Pages** — last 5 visited pages shown at the top when the search is empty (stored in `localStorage`)
- Full keyboard navigation: `↑`/`↓` to move, `Enter` to select, `Esc` to close
- shadcn `DialogContent` X button suppressed via `[&>button.absolute]:hidden` Tailwind selector to prevent overlap with the `Esc` hint

### 📤 Settings Export
- Export plugin configuration as a downloadable JSON bundle
- Sections included: Branding, Maintenance, SMTP/Email, Image Settings, Code Snippets, Redirects, Notes
- Bundle signed with an HMAC using the site's WordPress auth salt — tamper-evident
- Per-section checkboxes to choose which sections to include

### 📥 Settings Import
- Drag-and-drop or file-picker JSON import
- Client-side bundle preview — shows detected sections before applying
- Cross-site warning shown when HMAC signature does not match (import still allowed)
- Per-section overwrite control — choose exactly which sections to apply
- Import result summary after apply

### 🌐 WordPress XML Content Export
- Export site content as a standard WordPress WXR (XML) file without leaving the plugin
- Content type selector: **All**, Posts, Pages, Media, or any custom post type
- Triggers WordPress's built-in export generator (`export.php`) server-side

---

## REST API — New Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/settings/export` | Generate signed JSON settings bundle for selected sections |
| POST | `/settings/import` | Apply imported settings bundle with optional section filter |
| POST | `/settings/export-wp-xml` | Trigger WordPress native XML export and return download URL |

---

## Technical Notes

- Command Palette implemented as a React context provider (`CommandPaletteProvider`) + `useCommandPalette()` hook so any component can open it
- HMAC signed with `hash_hmac('sha256', $payload, wp_salt('auth'))`
- Settings import merges at the option level — does not overwrite unselected sections
- WordPress XML export uses `WP_Query` + WordPress core `export.php` include path

---

## Upgrade Notes

No database migrations required. The Command Palette is available immediately after activating the updated plugin.
