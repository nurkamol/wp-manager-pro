# WP Manager Pro v1.3.0

**Release date:** 2026-03-08

## Highlights

This release brings a fully redesigned Maintenance Mode with live preview and gradient customization, a new Security page for admin URL protection, AVIF image format support, and a completely upgraded Database Manager with full pagination and inline row editing — all with no extra dependencies.

---

## What's New

### 🎨 Maintenance Mode — Professional Appearance

The Maintenance page has been completely rebuilt with a two-column layout:

**Left column — tabbed settings:**
- **Content tab**: title, message, badge toggle + text, show/hide badge
- **Appearance tab**: 6 gradient presets (Midnight, Sunset, Forest, Royal, Slate, Candy), custom color pickers (start, end, accent, text), 8 emoji icon presets + custom emoji input
- **Extras tab**: countdown timer toggle + datetime picker

**Right column — live preview:**
- Updates in real-time as you change colors, icons, badge, and text
- Shows floating icon animation, accent divider, title, message, countdown placeholder blocks

**Save settings without toggling maintenance state** — use the "Save Settings" button to persist appearance without enabling/disabling the page.

### 🛡️ Security — Admin URL Protection

New **Security** page (sidebar shield icon) with:
- Replace `wp-login.php` with a secret URL slug of your choice
- Direct GET requests to `wp-login.php` are blocked and redirect to the homepage
- Login form POST, password reset, and other WP auth actions continue to work
- Copy-to-clipboard and open buttons for your new login URL
- One-click disable to restore default behavior

### 🖼️ Image Tools — AVIF Support

- New **AVIF Support** status indicator in the support cards row
- Toggle to enable AVIF uploads (`image/avif`, `image/avifs`)
- Requires PHP 8.1+ with GD (`imageavif` function) or ImageMagick compiled with AVIF codec
- Gracefully disabled when server doesn't support AVIF

### 🗃️ Database Manager — Pagination & Row Editing

The table data panel has been fully rebuilt:

- **Per-page selector**: 50 / 100 / 250 / 500 rows (backend limit raised to 500)
- **Full pagination**: first / prev / page X of Y / next / last with `rowStart–rowEnd of total` counter
- **Edit Row**: hover any row → pencil icon → Dialog with all columns pre-filled; primary key is read-only; shows column type hints
- **Delete Row**: trash icon → inline confirm (no extra modal) → row removed instantly
- **Add Row**: `+ Add Row` button in table header → Dialog; `auto_increment` fields auto-disabled with "auto-generated" placeholder
- **Column metadata**: backend now returns `primary_key` + `col_meta` per column (type, nullable, key, default, extra)

---

## Technical Notes

- **New REST routes**: `GET /security`, `POST /security/admin-url`, `DELETE /security/admin-url`, `POST /maintenance/settings`
- **New PHP controller**: `class-security-controller.php` — hooks into `login_init` (block) and `init` (custom URL redirect)
- **Maintenance template**: PHP generates a self-contained HTML page with inline styles — gradient background, glassmorphism icon container, floating keyframe animation, accent divider, conditional countdown JS timer
- **AVIF detection**: `function_exists('imageavif')` for GD; ImageMagick format list fallback
- **Dark mode**: All new pages (Maintenance, Security) fully support `.dark` class theming
