# v3.0.0 — Notification Centre + Major UI Refresh

**Released:** 2026-03-19

## Highlights

This is the largest feature release in WP Manager Pro's history. v3.0.0 adds six major capabilities centred around proactive monitoring, real-time UX polish, and developer productivity.

---

## New Features

### 1. Dark Mode Auto-Sync
- The plugin now follows the OS `prefers-color-scheme` media query when no manual light/dark preference has been saved.
- Three-state toggle in the sidebar: **Light** / **Dark** / **Auto** (follows OS).
- Preference persists in `localStorage`; the `dark` class is applied to `#wp-manager-pro-root` before first paint to avoid flash.

### 2. Dashboard Widgets
- Configurable widget grid on the Dashboard — show or hide any widget via the **Customize** button.
- Eight widgets: Quick Stats, Site Status, System Resources, Recent Posts, Quick Actions, Recent Audit Events, Cache Status, Uptime Ping.
- Widget visibility stored in `localStorage` per-browser.
- New REST endpoints: `GET /dashboard/uptime` (live HTTP ping to the site home URL), `GET /dashboard/widgets` (audit log + object cache snapshot).

### 3. Notification Centre
- Persistent bell icon in the sidebar header showing unread alert count.
- Slide-out notification panel (Radix UI Sheet) listing alerts with type icons, timestamps, and "Mark all read" / dismiss actions.
- REST endpoints: `GET /notifications`, `POST /notifications/read`, `DELETE /notifications/{id}`.
- Alert types: backup errors, cron failures, login lockouts, update failures, SSL expiry warnings, vulnerability alerts.

### 4. Mobile / Tablet Layout
- Fully responsive layout for phones (< 768 px) and tablets (768–1023 px).
- On mobile: sidebar is hidden; a **bottom nav bar** replaces it with Dashboard, Plugins, Themes, Database, and a "More" drawer showing all 28 pages in a 3-column grid.
- On tablet: sidebar auto-collapses to icon-rail mode.
- `useBreakpoint` hook drives all breakpoint-aware logic.
- Toaster repositioned to `top-center` on mobile.

### 5. Global Search
- The in-app Command Palette (`Cmd/Ctrl + Shift + P`) now searches live data as you type.
- Searches across plugins (name/description), WordPress users, notes, audit log entries, and `wp_options` keys.
- Results appear in a "Search Results" group below navigation items with a subtitle and type badge.
- Debounced 300 ms; minimum 2 characters; powered by new `GET /search?q=` endpoint.
- **System-wide Command Palette**: a standalone vanilla-JS overlay (`assets/global-palette.js`) is now injected into every WP admin page. The admin bar **WPMGR** button and the configured keyboard shortcut open this overlay from anywhere in `wp-admin` — not just inside the plugin.

### 6. Custom Post Type Manager
- Register, edit, and delete custom post types entirely from the UI — no code required.
- Fields: singular/plural labels (auto-slug), description, menu icon (dashicons picker + freetext), supports (title, editor, thumbnail, excerpt, custom fields, comments, revisions, author, page attributes), Public / REST API / Archive Page toggles.
- Separate **Taxonomies** tab to register custom taxonomies: singular/plural labels, Hierarchical / Public / REST toggles, attach to one or more post types (including built-in ones).
- All CPTs and taxonomies are registered at `init` priority 5 from `wp_options` storage — no plugin conflict.
- Guards against overwriting WordPress reserved slugs.
- REST endpoints: `GET/POST /cpt`, `DELETE /cpt/{slug}`, `GET/POST /cpt/taxonomies`, `DELETE /cpt/taxonomies/{slug}`, `GET /cpt/post-types`.

## UI / UX Improvements

- **Dark mode badge visibility** — all badge/pill colour maps updated with `dark:` variants across Audit Log, Content Tools, Snippets, Dev Tools, Developer Utilities.
- **Sheet & Dialog dark mode** — Radix UI portals now render inside `#wp-manager-pro-root` so they correctly inherit the `dark` class.
- **Tabs overflow** — TabsList gains `overflow-x-auto flex-nowrap scrollbar-none` for horizontal scrolling on narrow screens.
- **Admin bar button** — renamed to **WPMGR** with a search icon; tooltip shows keyboard shortcut.

## REST API

Nine new endpoints added in this release:

| Method | Path | Description |
|--------|------|-------------|
| GET | `/dashboard/uptime` | Live HTTP ping to site home |
| GET | `/dashboard/widgets` | Audit snapshot + cache stats |
| GET | `/notifications` | List notifications |
| POST | `/notifications/read` | Mark all notifications read |
| DELETE | `/notifications/{id}` | Dismiss a notification |
| GET | `/search` | Global search across plugins, users, notes, audit, options |
| GET | `/cpt` | List custom post types |
| POST | `/cpt` | Create / update custom post type |
| DELETE | `/cpt/{slug}` | Delete custom post type |
| GET | `/cpt/taxonomies` | List custom taxonomies |
| POST | `/cpt/taxonomies` | Create / update taxonomy |
| DELETE | `/cpt/taxonomies/{slug}` | Delete taxonomy |
| GET | `/cpt/post-types` | List all registered post types (for taxonomy picker) |
