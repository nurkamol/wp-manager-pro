## New REST API Endpoints (16 added)

```
POST   /plugins/upload        Upload plugin ZIP
GET    /plugins/export        Create plugin ZIP export
GET    /plugins/download      Stream plugin ZIP download

POST   /themes/upload         Upload theme ZIP
GET    /themes/export         Create theme ZIP export
GET    /themes/download       Stream theme ZIP download

POST   /files/upload          Upload file to directory
POST   /files/rename          Rename file or folder

POST   /database/row          Insert table row
PUT    /database/row          Update table row
DELETE /database/row          Delete table row
GET    /database/export       Export table as SQL dump

POST   /users/rename          Rename user login

GET    /reset/status          Get content counts
POST   /reset/execute         Execute site reset
```

## Plugin Highlights

### 🗜️ Plugin & Theme ZIP Import/Export
Upload a `.zip` to install or silently overwrite an existing plugin or theme — no uninstall required. Works through the "Upload ZIP" tab in each manager. Export packages any installed plugin or theme directory into a portable `.zip` ready for deployment elsewhere.

### 🖥️ Monaco Editor
The File Manager now uses the same editor engine powering VS Code. Language is auto-detected from the file extension: PHP, JavaScript, TypeScript, TSX, CSS, SCSS, JSON, SQL, YAML, HTML, XML, SVG, Markdown, Bash, ENV, INI, and more. The editor is fully integrated — save with the same Save button.

### 🗃️ Database Row CRUD
Click into any table row to edit it in-place, add a new row, or delete a row by its primary key. All three operations hit dedicated REST endpoints with proper sanitization and `$wpdb->prepare()`.

### 🖼️ SVG Support + Sanitization
Enable SVG uploads globally or restrict to specific roles (administrator, editor, author). Every SVG uploaded is sanitized server-side: `<script>` tags, `on*` event attributes, `javascript:` href values, `<foreignObject>`, and `<base>` elements are all stripped before the file is saved.

### 🗑️ Reset Tools
A new page in the sidebar lets you preview exactly how many posts, pages, comments, media files, and non-admin users exist before performing any action. Select the content types to remove, review the confirmation dialog, and confirm. Administrators are never affected.
